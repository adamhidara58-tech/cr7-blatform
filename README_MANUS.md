# خطة عمل تطوير نظام السحب ولوحة التحكم (Manus AI)

تم استلام الطلب من المستخدم لإصلاح لوحة التحكم وبناء نظام سحب متكامل مرتبط بـ NOWPayments.

## المهام المطلوبة:
1. **التوثيق (Authentication):** تنفيذ نظام JWT لحماية مسارات الأدمن.
2. **الإعدادات:** إصلاح مشكلة حفظ الإعدادات في لوحة التحكم.
3. **نظام السحب:**
    - سحب تلقائي للمبالغ ≤ 10$.
    - سحب يدوي للمبالغ > 10$.
    - دعم السحب الجماعي (Mass Payout).
4. **تحسينات الواجهة:** تطوير صفحة السحوبات بالفلاتر والإجراءات المطلوبة.
5. **قاعدة البيانات:** استخدام جداول Supabase الحالية (crypto_withdrawals, admin_settings, activity_logs).

## الخطوات التقنية:
- تحديث دوال Supabase Edge لمعالجة السحوبات والتحقق من الصلاحيات.
- تعديل واجهة تسجيل دخول الأدمن لاستخدام توثيق حقيقي بدلاً من localStorage فقط.
- ربط NOWPayments API بشكل آمن عبر متغيرات البيئة.
